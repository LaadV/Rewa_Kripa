Supabase Configuration

Supabase enables real-time seat sharing across all devices. Free tier is enough.
Without Supabase: bookings only visible on this device.

5-Minute Setup:
1. Visit supabase.com → Sign up (free, no credit card)
2. Create new project → choose nearest region
3. Left menu: SQL Editor → paste & run the SQL from README_SUPABASE.sql
4. Left menu: Settings → API → copy URL and anon/public key
5. Open supabase.js → replace YOUR_SUPABASE_URL and YOUR_SUPABASE_ANON
6. Re-upload the file → done! ✅

Change Admin Password

After changing, update ADMIN_PASSWORD in supabase.js on your server.

Data Management

Export creates a backup of all admin config. Clear removes all local storage (bookings + config).
