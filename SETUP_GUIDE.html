🗄️ Step 2: Set Up Supabase (10 Minutes, Free)

Create a free Supabase account

Go to supabase.com → Click Start your project → Sign up with GitHub or email (free, no credit card).

Create a new project

Click New Project → Give it a name like "rewa-kripa" → Choose the nearest region (Mumbai/Singapore for India) → Set a database password → Click Create new project.

💡 Wait 1-2 minutes for the project to initialize.

Run the database setup SQL

In your Supabase project → Left sidebar → SQL Editor → New Query → Paste the entire contents of README_SUPABASE.sql → Click Run.

When a customer sends a booking message on WhatsApp → Copy the message → Go to Admin → Seat Management → Paste in the "Parse WhatsApp Message" box → Click "Parse & Block Seats". Done!

Go to Admin → Buses/Routes/Trips → Edit details → Click "Save Bus/Route/Trip" → Click "Save All Changes" (top right). Changes are saved to Supabase and reflected across the website.

❓ Common Questions

What if Supabase is not connected? Will bookings still work? ▾
Yes! If Supabase is not configured or has an error, all bookings automatically save to the browser's localStorage. The seat map will still show booked seats — but only for the customer who made the booking on their device. Other customers won't see it. This is why Supabase setup is recommended.
The admin password isn't working. How do I reset it? ▾
Open supabase.js in a text editor. Find the line: const ADMIN_PASSWORD = 'Swift@8606'; Change 'Swift@8606' to your new password. Save and re-upload the file to your server.
Bookings are not showing up in the seat map. Why? ▾
1. If Supabase is connected: check that the SQL was run correctly (Step 3 above). 2. If using localStorage only: bookings are per-device. Only the device that made the booking sees it. 3. Make sure you're looking at the correct bus and date in the seat map.
How do I add more buses or change prices? ▾
Go to admin.html → Buses & Drivers (sidebar) → Select the bus tab → Edit price, capacity, etc. → Click "Save Bus" → Then click "Save All Changes" top right. Changes take effect immediately.
The Finance Manager is not saving data after page refresh. Why? ▾
Finance data is saved in browser localStorage by design (it's private business data). It will persist between normal page refreshes. However, if you clear your browser data or use a different browser/device, it will be gone. Regularly export your data using the "Export Excel" button as a backup.
